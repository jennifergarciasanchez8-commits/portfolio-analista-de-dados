# Análise de Dados de um Salão de Unhas

## Sobre o projeto

Este projeto foi desenvolvido com dados de um salão de unhas, com o objetivo de organizar, analisar e apresentar informações importantes sobre o desempenho do negócio.

A análise permite compreender melhor o faturamento, os serviços realizados, as formas de pagamento e outros indicadores relevantes para apoiar a tomada de decisões.

## Objetivos

- Organizar e limpar os dados do salão.
- Identificar os principais serviços realizados.
- Analisar o faturamento do negócio.
- Avaliar as formas de pagamento utilizadas.
- Criar indicadores e visualizações.
- Desenvolver um dashboard no Excel.

## Ferramentas utilizadas

- Microsoft Excel
- Fórmulas e funções
- Tabelas dinâmicas
- Gráficos
- Segmentação de dados
- Dashboard

## Etapas do projeto

1. Importação e organização dos dados.
2. Limpeza e padronização das informações.
3. Criação de cálculos e indicadores.
4. Desenvolvimento de tabelas dinâmicas.
5. Criação de gráficos.
6. Construção do dashboard.
7. Análise dos resultados.
