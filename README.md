### 1. Análise de dados de um salão de unhas

Projeto desenvolvido para organizar e analisar dados de um salão de unhas, com o objetivo de identificar informações importantes sobre faturamento, serviços realizados, formas de pagamento e desempenho do negócio.

**Ferramentas utilizadas:**
- Excel
- Tabelas dinâmicas
- Gráficos
- Dashboard

**Principais etapas:**
- Organização e limpeza dos dados
- Criação de cálculos e indicadores
- Desenvolvimento de tabelas dinâmicas
- Criação de gráficos
- Construção de um dashboard no Excel

**Objetivo do projeto:**

Transformar dados brutos em informações visuais e úteis para apoiar a tomada de decisões do salão.
